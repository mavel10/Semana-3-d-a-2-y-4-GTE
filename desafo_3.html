Traducción y Clasificación de Frases Cortas con la Máquina Educable.

Entrenar un modelo en la Máquina Educable para clasificar frases cortas en tres categorías como preguntas, negaciones, y afirmaciones. Solicitar al usuario que ingrese al menos diez frases para cada categoría usando el teclado de su dispositivo, usar estas frases para entrenar el modelo, luego, diseñar un programa que permita al usuario escribir una nueva frase y clasificarla con el bloque de predicción, utilizando los bloques de traducción para traducir la categoría detectada al idioma seleccionado por el usuario como inglés francés o portugués, mostrar en pantalla la categoría de la frase clasificada y su traducción, solicitar al docente que pruebe el sistema ingresando frases diferentes para verificar si las clasificaciones son correctas.
